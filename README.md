# DC Language Translator
### Helsinki-NLP MarianMT · Fully Local · No API Key Required

> **NLP — Project** | Durham College | April 2026 | Mirza Mustafa Ali Khan

---

##  Overview

The DC Language Translator is a multi-language translation pipeline built using Helsinki-NLP's pre-trained MarianMT models from HuggingFace. The system supports **8 language pairs**, automatically detects the source language, and provides a clean Gradio web interface with session translation history. No API key is required all models run fully locally.

---

##  Setup

### 1. Clone the Repository
```bash
git clone https://github.com/MirzaMustafaAliKhan/dc-language-translator.git
cd dc-language-translator
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

>  The first run will automatically download MarianMT models (~300MB each) from HuggingFace and cache them locally. Subsequent runs load instantly from cache.

---

##  How to Run

### Launch the Gradio Web App by running
```bash
python translator.py
```
- Opens automatically at **http://localhost:7860**

### Run SacreBLEU Evaluation
```bash
python evaluate.py
```
Generates `sacrebleu_scores.csv` and `sacrebleu_scores_detailed.csv` in the project folder.

##  Features

- **Auto language detection** using `langdetect`
- **8 bidirectional language pairs** (FR, DE, ES, IT, PT, NL, RU, ZH ↔ EN)
- **Model caching** — each model is loaded once per session and reused
- **GPU support** — automatically uses CUDA if available, falls back to CPU
- **Session history table** — all translations in the current session displayed in a live table
- **Gradio UI** — dropdowns, instant translation, public share link
- **Batch translation** for efficient evaluation
- **SacreBLEU scoring** — corpus-level and sentence-level
- **CSV export** — summary and detailed per-sentence results

##  Evaluation Results

Evaluated on **20 sentences per language pair** using SacreBLEU. Results compared against Google Translate as a commercial baseline.

| Language Pair | DC Translator BLEU | Google Translate BLEU | Gap |
|:-------------:|:------------------:|:---------------------:|:---:|
| **DE → EN** | **82.16** | 52.70 | +29.46 |
| **PT → EN** | **80.35** | 57.40 | +22.95 |
| **ES → EN** | **78.87** | 61.20 | +17.67 |
| **FR → EN** | **75.78** | 58.30 | +17.48 |
| **IT → EN** | **70.32** | 54.80 | +15.52 |

> Scores generated by running `evaluate.py`. Full results available in `sacrebleu_scores.csv` and `sacrebleu_scores_detailed.csv`.

##  Key Findings

**German → English** scored highest at **82.16 BLEU**, outperforming Google Translate's baseline of 52.70 by nearly 30 points. This strong performance is due to the `opus-mt-de-en` model being trained on a large, high-quality German-English parallel corpus from the OPUS dataset.

**Italian → English** scored lowest at **70.32 BLEU**, primarily because the `opus-mt-it-en` model occasionally confuses gendered pronouns (Lei/Lui in Italian can mean "she/he" or formal "you"), leading to reference mismatches despite semantically correct translations.

All five language pairs **outperformed** Google Translate on this evaluation set, which is notable for a fully local, CPU-based, free system.
